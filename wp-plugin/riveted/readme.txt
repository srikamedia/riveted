A Google Analytics plugin for measuring active time on site.

== Description ==

This plugin monitors whether a user is active and pings Google Analytics every 5 seconds to record the amount of active time spent on the page. It supports Universal Analytics, Classic Google Analytics, and Google Tag Manager. It has no other dependencies.

== Installation ==

1. Upload the `riveted` directory into the `/wp-content/plugins/` directory on your server.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. If you'd like to modify the default settings, go to the Riveted panel under Settings.
